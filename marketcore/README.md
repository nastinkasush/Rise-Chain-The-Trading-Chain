# RISE MarketCore

**RISE MarketCore** — це нативна інфраструктура orderbook'ів на рівні протоколу.

### Ключові можливості:
- Shared liquidity (спільна ліквідність для всіх додатків)
- Дозволяє будь-кому створювати spot, perpetuals та інші ринки
- Повна composability з EVM
- Низька затримка та висока швидкість виконання

MarketCore — основа для всіх торгових додатків в екосистемі Rise Chain.

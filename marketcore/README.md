# RISE MarketCore

**RISE MarketCore** — это нативная инфраструктура orderbook'ов на уровне блокчейна.

### Ключевые возможности:
- Общая ликвидность для всех приложений
- Возможность создавать spot, perpetuals и другие рынки
- Полная совместимость с EVM
- Очень высокая скорость исполнения ордеров

MarketCore — основа для всех торговых проектов в экосистеме.

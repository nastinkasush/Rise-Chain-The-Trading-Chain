## Что такое Rise Chain?

**Rise Chain** — это Ethereum Layer 2, специально созданный для финансовых приложений и on-chain трейдинга.

- **1 мс** latency
- **50 000+ TPS**
- Нативные **orderbook'ы** (MarketCore)
- Полная EVM-совместимость
- Secured by Ethereum

Rise — это не просто ещё один L2. Это **блокчейн, где orderbook — это примитив первого класса**, а все DeFi-приложения могут взаимодействовать с ним в рамках одной транзакции.

### Ключевые фичи

- **Continuous Block Pipeline** + **Secured Shred Preconfirmations**
- **pEVM** (Parallel EVM) — параллельное исполнение транзакций
- **RISEx** — встроенная unified exchange с on-chain perpetuals
- **AutoYield** — автоматический yield на idle margin
- Portfolio margining (всё портфолио работает как одно обеспечение)
- Поддержка крипты, акций, commodities 24/7

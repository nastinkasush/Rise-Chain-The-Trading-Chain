# RISE Chain (Ядро)

**RISE Chain** — это высокопроизводительный Ethereum Layer 2, специально созданный для on-chain трейдинга и финансовых приложений.

### Основные характеристики:
- Параллельный EVM (pEVM)
- Задержка около 1 мс
- Производительность более 50 000 TPS
- Защищён Ethereum
- Нативные orderbook'и на уровне протокола

Это фундамент всей экосистемы Rise Chain.

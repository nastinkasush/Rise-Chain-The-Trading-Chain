# RISEx

**RISEx** — главная on-chain биржа perpetuals на Rise Chain.

### Основные функции:
- Обновление orderbook каждую 1 миллисекунду
- AutoYield — автоматический доход на свободную маржу
- Portfolio Margining — весь портфель как единое обеспечение
- Торговля криптовалютой, акциями и товарами 24/7
- Интерфейс уровня CEX при полной децентрализации
